# vending-machine-system

## Description
A Java program that simulates a real-world vending machine: takes user selections, processes/manages them, handles payments, and calculates change.

## Classes
- Buying_A_Snack.java: handles snack purchases
- Item.java: represents snack items
- VendingMachine.java: main logic
- VendingMachineStocker.java: manages inventory

## How to Run
1. Clone the repo
2. Compile all `.java` files
3. Run `Main.java`
